# Recipe: Sequence Patterns

Task: choose the right stimulus structure quickly.

## Directed

Use fixed vectors for bring-up and deterministic debugging.

## Random

Use randomized fields per item for broad exploration.

## Exhaustive (small spaces)

Nested loops for all combinations of low-bitwidth inputs.

## Composed scenario

Chain helper operations into algorithmic flows.

## Rule of thumb

Start directed, add random, then retain a small deterministic smoke subset for CI.
